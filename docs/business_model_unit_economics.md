# Amaswarn: Business Model & Unit Economics

## 1. Modèle de Revenus (Subscription Based)
- CAC (Customer Acquisition Cost) : Ciblage via GitHub Marketplace et intégrations Slack.
- LTV (Lifetime Value) : 18-24 mois estimé du fait de l'intégration profonde dans le workflow (Sticky product).

## 2. Structure de Coûts (Unit Economics)
- Calcul Dérivation Coût Agent :
  - Moyenne 5-10 tokens par Issue (RAG + Planning + Implement).
  - Coût moyen LLM (GPT-4o) : 0.05€ à 0.15€ par PR.
  - Vente PR réussie (Pricing implicite) : 5€ à 10€ de valeur générée.
- Marge brute estimée : > 85% (après coûts Cloud & API).

## 3. Tiering & Packaging
| Feature                | Dev (Free) | Team (99€) | Enterprise (Quote) |
|------------------------|------------|------------|--------------------|
| Autonomous PR generation| No         | Yes        | Yes                |
| Self-healing CI        | No         | Yes        | Yes (Unlimited)    |
| Custom Policy Engine   | No         | Basic      | Advanced (RBAC)    |
| SOC2 Audit Logs        | No         | No         | Yes                |
| Multi-Cloud Support    | No         | No         | Yes                |

## 4. Stratégie de Croissance (The Flywheel)
1. L'IA résout des issues simples -> Gain de confiance du CTO.
2. Déblocage du Self-healing CI -> Dépendance structurelle gagnée.
3. Adoption du Policy Engine -> Expansion Enterprise totale.

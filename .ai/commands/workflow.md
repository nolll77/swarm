# AI Commands Workflow

## 📋 PLAN
```md
Analyze the following task.

1. Restate the problem
2. Identify constraints
3. Break it into steps
4. Propose a clean architecture approach
5. List risks

Do NOT write code yet.
```

## ⚙️ IMPLEMENT
```md
Implement the solution step by step.

Requirements:
- Follow project conventions
- Write clean, maintainable code
- Handle edge cases
- Add minimal comments

Explain your reasoning before coding.
```

## 🧪 TEST
```md
Write tests for the implemented feature.

Include:
- Normal cases
- Edge cases
- Failure scenarios

Explain what each test validates.
```

## 🔍 REVIEW
```md
Review this code like a senior engineer.

Check:
- Bugs
- Edge cases
- Performance issues
- Readability

Suggest improvements.
```
